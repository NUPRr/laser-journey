# Laser Journey

Laser Journey is a full-stack web application designed to simplify the management of laser hair removal treatments. It provides users with a centralized platform to monitor treatment progress, schedule appointments, track completed sessions, and maintain their treatment history in a clean and intuitive interface.

## Live Demo

https://laser-journey.vercel.app/

## About the Project

Managing multiple laser treatment sessions can become difficult over time, especially when appointments, progress, and treatment areas need to be tracked consistently. Laser Journey was built to solve this problem by providing an organized digital solution that helps users stay informed throughout their treatment journey.

The application focuses on usability, responsive design, and secure user authentication while delivering a modern user experience.

## Features

- Secure user registration and login using JWT authentication
- Personalized dashboard for every user
- Track laser treatments for multiple body areas
- Monitor completed treatment sessions
- Appointment scheduling and upcoming appointment reminders
- Treatment progress visualization
- Session history management
- Responsive design for desktop, tablet, and mobile devices
- Clean and intuitive user interface
- RESTful API integration between frontend and backend
- Persistent MySQL database storage
